# Shopping List App

A simple shopping list web application where you can create lists and manage your shopping items together.

## üõí Features

- **Create & Share Lists** - Generate unique list IDs to share with anyone
- **Real-time Collaboration** - Multiple people can add, check, and remove items
- **Simple Interface** - Clean, minimal design focused on usability
- **Local Storage** - All data saved in your browser
- **Item Management** - Add, check off, and delete items easily
- **Bulk Actions** - Clear all checked items or clear entire list at once

## üéØ How to Use

1. **Create a List** - Click "Create New List" to start a new shopping list
2. **Share the ID** - Copy the list ID and share it with family or friends
3. **Join a List** - Enter a list ID to access a shared shopping list
4. **Add Items** - Type items and click "Add" or press Enter
5. **Check Off Items** - Click the checkbox as you shop
6. **Manage Items** - Delete individual items or clear checked/all items

## üöÄ Getting Started

1. **Download the files** - Save all 5 files in the same folder:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `ui.js`
   - `storage.js`

2. **Start a local server** - Choose one of these methods:

   **Using Python:**
   ```bash
   # Python 3
   python -m http.server 8080
   # Python 2
   python -m SimpleHTTPServer 8080
   ```

   **Using VS Code:**
   - Install the "Live Server" extension
   - Right-click `index.html` and select "Open with Live Server"

3. **Open in browser** - Go to `http://localhost:8080`

4. **Start shopping** - Create your first list and start adding items!
